# FastAPI使用教程

## 简介

- python web框架，基于Starlette（异步框架）和Pydantic（数据验证库）构建，具有高性能、易用性和可扩展性。
- 使用uvicorn(ASGI服务器)，支持高并发请求。
- 自动生成API文档，内置Swagger UI和ReDoc。

## 服务器

- CGI 通用网关接口
- WSGI Web服务器网关接口， 针对Python生态优化，取代CGI，专注于同步web应用
- ASGI 异步服务器网关接口，WSGI的升级，适应异步编程和现代web需求（websocket、http2），兼容wsgi应用。

## 虚拟环境

创建虚拟环境

```bash
conda create -n fastapi_env python=3.12   
```

检查虚拟环境是否激活
```bash
which python
```

安装fastapi和uvicorn
```bash
pip install "fastapi[standard]"  
```

启动服务
```bash
 uvicorn main:app 
```

热更新
```bash
 uvicorn main:app --reload 
```

调试模式
```bash
fastapi dev main.py
```

通过py运行： `python main.py`，需要在main.py中添加以下代码：

```python
import uvicorn
if __name__ == "__main__":
    uvicorn.run("main:app", host="0.0.0.0", port=8000, reload=True)
```
